How it works (no tricks)

Every day at 00:00 UTC a new river is generated. Every racer's run is computed from the day plus its name — nothing else. Same name, same day, same finish, on any phone on Earth. That's why a result can be checked by anyone you send it to: they type the name, they see the run.
